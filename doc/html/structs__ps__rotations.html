Detailed Description

Base for typedef s_ps_rotations to storage the number of each rotation instruction type to be performed on the stacks.

This type is used to storage the number of each rotation instruction type to be performed on either the stack B or stack A. This values are set, in respect to the lowest-cost element in B so that rotate_before_pa() execute the right instructions before the actual pushing to A of said element.
