Detailed Description

Support functions for the "push swap language". Memory allocation and freeing function for the structure; initialization of the structure with user arguments; no-duplication check; etc.

bottom_pusha – Pushes an integer to the bottom of stack a.

Pushes an integer to the bottom of stack a. This function is used to initialize the content from the command-line arguments according to this requirement: "The first argument should be at the top of the stack (be careful about the order)".

Parameters
[in]ps- The address of the pointer to the push swap stacks struct.
[in]element- The integer to push to the bottom of stack a.
Returns
- A non-zero value if succesful.
A 0 value if memory allocation fails.
