Detailed Description

Set of function to interpret push-swap instruction language read from a file descriptor (one instruction per line). The instructions will be executed on an already initialized t_ps_stacks structure containing both a and b stacks.

ps_lang_interpreter – Read and executes push-swap instructions

Reads a line from a file descriptor looking for valid push-swap instructions. The instructions are executed in the order in which were read until an error is found or the end of the file is reached.
The t_ps_stacks data structure passed must be already initialized with the desired data and is modified accordingly to the instructions.

Parameters
[in,out]ps- Pointer to a t_ps_stacks containing the pair of stacks with valid data.
[in]fd- A valid file descriptor from where the single instruction lines will be read from.
[in]debug- Non-zero value if extra output is to be printed for debugging purposes.
Returns
Non-zero value if a valid instruction was parsed and executed.
Returns 0 if no valid instruction was found.
